AHORA SÓ LO NECESITAS:
═══════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALAR DEPENDENCIA PYTHON:
    
    pip install shapely lxml

2️⃣  EN FRONTEND, IMPORTAR COMPONENTE:
    
    En tu router/página, agregar:
    
    import CoverageChecker from '@/components/CoverageChecker.vue'
    
    Y en rutas:
    {
      path: '/coverage',
      component: () => import('@/components/CoverageChecker.vue'),
      meta: { requiresAuth: true }
    }

3️⃣  EN EL MENÚ, AGREGAR LINK:
    
    <router-link to="/coverage">🗺️ Consulta de Cobertura</router-link>

4️⃣  INICIA BACKEND Y FRONTEND:
    
    Terminal 1:
    cd backend && npm start
    
    Terminal 2:
    cd frontend && npm run dev

5️⃣  ABRE EN NAVEGADOR:
    
    http://localhost:5173/coverage
